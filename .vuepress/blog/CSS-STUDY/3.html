# 元素居中方案

# 水平居中

  • 行内元素(inline)
/* 设置父级元素 */
.inline-parent {
    text-align: center;
}
  • 块级元素(block)
    • 常规
    .parent {
    width: 300px;
    background: lightgreen;
}
.element {
    margin: 0 auto; 

    width: 100px;
    height: 100px;
    background: lightyellow; 
}
    • 绝对定位
    // 绝对定位
.parent {
    position: relative;

    height: 100px;
    background: lightgreen;
    width: 300px;
}
.element {
    position: absolute;
    left: 50%;
    transform: translate(-50%, 0); 
    /* 同理这里可以使用 margin-left: -50px;代替 translate*/

    width: 100px;
    height: 100px;
    background: lightyellow; 
}

// 绝对定位方案二
.element {
    position: absolute;
    left: 0;
    right: 0;
    margin: 0 auto;

    width: 100px;
    height: 100px;
    background: lightyellow; 
}
    • display: flex;
    .parent {
    display: flex;
    justify-content: center;
    height: 100px;
    background: lightgreen;
    width: 300px;
}
.element {
    width: 100px;
    height: 100px;
    background: lightyellow; 
}

# 垂直居中

  • 行内元素
.parent {
    height: 100px;
}
.element {
    line-height: 100px;
}
  • 块级元素
    • inline-block
    .parent::after, .element{
    display:inline-block;
    vertical-align: middle;
}
.parent::after{
    content:'';
    height:100%;
}
    • table
    .parent {
    display: table;
}
.element {
    display: table-cell;
    vertical-align: middle;
}
    • flex
    .parent {
    display: flex;
    align-items: center;
}
    • 绝对定位
    .parent {
    position: relative;
}
.element {
    position: absolute;
    top: 50%;
    transform: translate( 0, -50%);
    /* 同理可以用margin-top: -0.5 * element高度来代替 */
}
// 方案二
.element {
    position: absolute;
    top: 0;
    bottom: 0;
    margin: auto 0;
}

从上面可以看出 flex 和 绝对定位 都可以应付水平和垂直居中
